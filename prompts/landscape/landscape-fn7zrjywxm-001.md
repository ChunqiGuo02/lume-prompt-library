# 正文 prompt
请为【【奶粉】】生成一张“手绘包装提案板风格”的产品介绍海报。 

如果我提供了产品图片，请以该图片为主要参考，保留产品的核心外观特征、包装形态、颜色、材质、品牌识别和整体气质，并优先沿用产品图中的主色调与配色关系进行设计。
如果我没有提供产品图片，请根据【【奶粉】】自动生成符合该产品定位的产品外观与包装设计，并根据产品属性、品类特征和气质，自动搭配自然合理的主色、辅助色和背景色。

海报整体风格要求：
这不是普通电商详情页，也不是单纯棚拍海报，而是一张融合“产品成品图 + 包装设计草图 + 品牌概念开发过程”的创意提案板。整体呈现为高审美、高完成度的品牌设计展示图，像设计师正在展示一个包装概念方案。

画面要求：
- 竖版构图
- 使用统一且有品牌感的主色调背景，背景颜色根据产品本身自然决定，不固定为某一种颜色
- 配色应与产品属性协调：如果有参考图，优先继承参考图的主色和品牌色；如果没有参考图，则根据产品名称自动生成合理的色彩方案
- 画面中央放置一个最完整、最吸睛的产品主视觉，具有真实质感、立体光影和商业成品效果
- 产品周围加入多个辅助视觉元素：不同角度的包装草图、结构草图、局部造型探索、手持展示小稿、包装展开思路、品牌字样排版试验
- 加入自然随性的黑色或深色手绘线稿、箭头、圈画、符号、手写中文批注，让画面具有强烈“设计过程感”和“创意提案感”
- 成品图要相对精致真实，草图部分要相对随性灵动，形成“完成品 + 设计草图”的对比
- 版式看起来松弛自由，但整体要有秩序、有节奏、有视觉中心，体现高级平面设计感
- 可以适度加入与产品相关的小图形、小符号或趣味元素，增强品牌记忆点
- 最终效果应具有品牌感、创意感、过程感和视觉冲击力

配色原则：
- 不要固定黄色背景
- 要根据产品类型、材质、用途、消费感受和品牌气质自动选择主色调
- 背景色、产品色、文字色、草图线稿色应彼此协调
- 可以使用高饱和单色背景，也可以使用柔和统一色系背景，只要整体有品牌感和视觉冲击力
- 色彩应自然服务于产品，而不是喧宾夺主

风格关键词：
包装设计提案板、品牌概念开发、手绘草图、设计过程感、moodboard、创意评审稿、产品成品渲染、视觉实验、商业设计海报

请特别注意：
- 如果有参考产品图，优先遵循参考图，不要偏离产品本身
- 如果没有参考图，则根据产品名称自动生成合理的产品外观、包装和配色方案，并保持整体风格统一
- 画面重点是“产品介绍 + 创意设计过程展示”
- 不要做成普通电商详情页，不要做成促销海报，不要过度整齐呆板，不要缺少草图和手写批注，不要廉价感，不要低幼卡通感

请为 【奶粉】 生成一张“手绘包装方案项目风格”的产品介绍海报。

如果我提供了产品图片，请将其作为主要参考，保留产品的核心视觉特征、包装形式、颜色、材质、品牌标识和整体气质。设计时优先考虑产品图片中的主色调和色彩关系。
如果我没有提供产品图片，请根据 【奶粉】 的定位自动生成产品外观和包装设计，并根据产品属性、品类特征和气质自动匹配自然且合理的背景色、主色和辅助色。

这不是标准的电商详情页或简单的摄影棚拍摄海报，而是一个融合了“产品成品图 + 包装设计草图 + 品牌概念开发过程”的创意项目。整体呈现为一张高审美、高完成度的品牌设计展示图，如同设计师正在展示一份包装概念方案。

- 竖构图。
- 使用具有品牌感的统一背景；背景色应由产品本身自然决定，不固定为任何单一颜色。
- 配色方案应与产品属性协调：如有参考图，优先继承其主色和品牌色；如无，则根据产品名称自动生成合理的配色方案。
- 将最完整、最引人注目的产品主视觉放置在中心，具备真实的质感、3D 灯光和商业级成品效果。
- 在产品周围环绕多个辅助视觉元素：不同角度的包装草图、结构草图、局部形态探索、手持展示草图、包装展开构思以及品牌字体排版实验。
- 添加自然随意的黑色或深色手绘线条、箭头、圆圈、符号和手写中文注释，使画面具有强烈的“设计过程感”和“创意方案感”。
- 成品图部分应相对精致写实，而草图部分应相对随意灵动，形成“成品 + 设计草图”的对比。
- 布局看起来应轻松自由，但整体必须有秩序、有节奏感并具备视觉中心，体现出高端平面设计感。
- 可适当添加与产品相关的小图形、符号或趣味元素，以增强品牌记忆点。
- 最终效果应具备品牌感、创意感、过程感和视觉冲击力。

色彩原则：
- 不要固定使用黄色背景。
- 根据产品类型、材质、用途、消费体验和品牌气质自动选择主色调。
- 背景色、产品色、文字色和草图线条色应相互协调。
- 可以使用高饱和度的单色背景或柔和统一的配色方案，只要整体画面具有品牌感和视觉冲击力即可。
- 颜色应自然地服务于产品，而不是喧宾夺主。

风格关键词：
包装设计方案项目，品牌概念开发，手绘草图，设计过程感，情绪板，创意评审草稿，产品成品渲染，视觉实验，商业设计海报。

特别说明：
- 如有参考产品图，优先遵循参考图，不得偏离产品本身。
- 如无参考图，根据产品名称自动生成合理的产品外观、包装和配色方案，保持整体风格一致性。
- 图像的重点是“产品介绍 + 创意设计过程展示”。
- 不要将其制作成普通的电商详情页、促销海报，或显得过于整齐僵硬；不可缺少草图和手写注释；避免廉价感或不成熟的卡通风格。
