# 正文 prompt
【MODE（必ず1つだけ選ぶ／複数禁止）】
MODE: {argument name="Mode" default="A"} / B / C / D
•A＝被写体強調（主役を一目で際立たせる）
•B＝空間没入（奥行きと臨場感を強める）
•C＝空気感（湿度・温度・重さなど“空気”を主役に）
•D＝映画的完成度（派手さより完成版らしさ優先）

⸻

【根本目的】
添付画像を映画制作の最終ポストプロダクション工程として扱い、
レンズ・光・特殊効果・カラーを用いて「一目で変化が分かる完成版カット」に仕上げる。
遠慮は不要。ただし工程と責任を明確にし、やりすぎは避ける。

⸻

【開始フレーム（絶対）】
添付画像をそのまま使用。
構図、被写体、配置、向き、比率を一切変更しない。
カメラ位置も変更しない。トリミング禁止。
新しい物体や文字や装飾を追加しない。除去もしない。

⸻

【固定条件（絶対遵守／変更禁止）】
時間帯・天候・季節・物語状況は不変。
これらを誤認させる演出は禁止。
演出してよいのは「見え方」だけ。

⸻

【工程1：レンズ再設計（必ず“構造的変化”を作る）】

元画像のレンズ特性（焦点距離印象／被写界深度／周辺描写）を解析し、
映画用レンズの物理特性へ明確に変更する。
以下を必ず再設計する：
•焦点距離の印象
•被写界深度
•周辺描写（ボケ、減光、収差）

MODE別レンズ方針（必ず従う）
•MODE A：主被写体の分離を最優先。背景整理。浅すぎ禁止だが“主役だけが明確”な深度へ。軽い周辺減光で視線を中心へ。
•MODE B：空間の奥行きを最優先。前景〜遠景の距離感が読み取れる深度へ。周辺情報は残し、空間を途切れさせない。
•MODE C：空気の層を最優先。標準域の自然な印象。階調と微細な滲みで密度を感じさせる（ただし濁り禁止）。
•MODE D：映画的安定感を最優先。極端なボケや誇張は禁止。収差は抑え、品位あるレンズ感に統合。

この工程の合格条件：
「元画像と比べて、画の立体構造が変わった」と一目で分かること。

⸻

【工程2：ライティング再構築（視線誘導）】

光源の追加・強調は許可。
ただし新しい光は“物語的に存在し得る範囲”のみ。
時間帯・天候を誤解させる光は禁止。
光は必ず主被写体への視線誘導に使う。

MODE別ライティング方針
•MODE A：主被写体のキー/リム/反射を整理して存在感を上げ、背景は抑える。
•MODE B：前景→中景→背景へ光のグラデーションを作り、奥行き方向に導く。
•MODE C：拡散光と回り込み光を重視し、空間全体の密度を上げる（光源感は強めすぎない）。
•MODE D：不自然なハイライトは禁止。既存光の“整理”を優先し、自然な映画照明に整える。

⸻

【工程3：特殊効果（空間の触感）】

目的は派手さではなく「空間に触れられる感覚」。
使用許可：ボリュームライト／空気中粒子／霧・霞・湿度／控えめレンズフレア／光散乱。
効果は均一にかけず、主被写体を邪魔しない。

MODE別投入量
•MODE A：最小限。主役周囲と視線の通り道だけ。
•MODE B：中程度。奥行き方向の層を作る。
•MODE C：微量だが全体に“密度”として存在（目立たせない）。
•MODE D：見えないレベル。存在しても気付かない品位。

⸻

【工程4：カラーグレーディング（役割の再配置）】

色相は維持。色を足さず、役割を明確にする。
彩度・コントラスト・明度を再配置し、映画的トーンへ統合する。

MODE別カラー方針
•MODE A：主役の明瞭度と分離を最優先。背景は彩度/情報を抑える。
•MODE B：奥行き方向の色の層で空間を支える。グラデーション重視。
•MODE C：彩度は控えめに、明度差と微細な色温度で空気を表現。
•MODE D：LUT感禁止。統一感と品位を最優先し、過度な主張は避ける。

⸻

【工程5：最終チェック（必須）】

以下をすべて満たせ：
•元画像と並べて一目で違いが分かる
•状況が変わったとは感じない（時間帯・天候・季節・物語は不変に見える）
•光と特殊効果がやりすぎに見えない
•1カットの映画として成立する
•超高精細。映画品質。

## 中文译文（源站提供）
【模式 (只能选择一个 / 禁止多选)】
模式: {argument name="Mode" default="A"} / B / C / D
•A = 主体突出 (让主要主体一目了然)
•B = 空间沉浸 (增强深度和真实感)
•C = 氛围感 (关注湿度、温度、重量等“空气感”)
•D = 电影级完成度 (优先考虑完成质量而非华丽程度)

⸻

【基本目标】
将附带图像视为电影制作的后期制作阶段，使用镜头、光线、特效和色彩，将其完成为“一眼就能看出明显变化的完成剪辑”。
无需保留。但是，请明确过程和责任，避免过度处理。

⸻

【起始帧 (绝对)】
按原样使用附带图像。
不得改变构图、主体、位置、方向或比例。
不得改变摄像机位置。禁止裁剪。
不得添加或删除新对象、文本或装饰。

⸻

【固定条件 (绝对遵守 / 禁止更改)】
时间、天气、季节和故事情境保持不变。
禁止误导这些的导演。
只能指导“外观”。

⸻

【流程 1: 镜头重新设计 (必须创建“结构性变化”)】

分析原始图像的镜头特征 (焦距印象 / 景深 / 周边描绘)，并将其明确更改为电影镜头的物理特征。
必须重新设计以下内容:
•焦距印象
•景深
•周边描绘 (散景、暗角、像差)

各模式的镜头策略 (必须遵循)
•模式 A: 优先分离主要主体。背景组织。禁止浅景深，但要向“只有主要主体清晰”的景深移动。轻微暗角引导视线到中心。
•模式 B: 优先空间深度。向前景到远背景的距离可读的景深移动。周边信息保留，空间不中断。
•模式 C: 优先空气层。标准范围内的自然印象。通过渐变和微妙模糊感受密度 (但禁止浑浊)。
•模式 D: 优先电影稳定性。禁止极端散景或夸张。抑制像差，镜头感与庄重感融为一体。

此流程的通过条件:
与原始图像相比，图像的三维结构发生变化必须一目了然。

⸻

【流程 2: 光照重建 (视线引导)】

允许添加和强调光源。
但是，新光源必须仅在“叙事上合理”的范围内。
禁止误导时间或天气的光线。
光线必须始终用于引导视线到主要主体。

各模式的光照策略
•模式 A: 组织主要主体的关键光/轮廓光/反射光以增强存在感，并抑制背景。
•模式 B: 创建从前景 → 中景 → 背景的光线渐变，在深度方向引导视线。
•模式 C: 强调漫射光和环绕光，以增加整个空间的密度 (不要过度增强光源感)。
•模式 D: 禁止不自然的强光。优先“组织”现有光线，并将其调整为自然的电影光照。

⸻

【流程 3: 特效 (空间的触感)】

目的不是华丽，而是“能够触摸到空间的感觉”。
允许使用: 体积光 / 空中颗粒 / 薄雾、雾霾、湿度 / 微妙的镜头光晕 / 光散射。
效果不应均匀应用，也不应干扰主要主体。

各模式的输入量
•模式 A: 最小。仅在主要主体周围和视线路径上。
•模式 B: 中等。在深度方向创建层次。
•模式 C: 微量，但以“密度”形式存在于整个空间 (不要使其突出)。
•模式 D: 不可见的水平。即使存在也察觉不到的庄重感。

⸻

【流程 4: 色彩分级 (角色重新分配)】

色相保持不变。不添加颜色，但明确角色。
重新分配饱和度、对比度和亮度，并整合到电影色调中。

各模式的色彩策略
•模式 A: 优先主要主体的清晰度和分离。抑制背景的饱和度/信息。
•模式 B: 通过深度方向的色彩层次支持空间。强调渐变。
•模式 C: 适度饱和，通过亮度和微妙色温差异表达空气感。
•模式 D: 禁止 LUT 感。优先统一性和庄重感，避免过度强调。

⸻

【流程 5: 最终检查 (必需)】

满足以下所有条件:
•与原始图像并排比较时，差异一目了然
•不觉得情境发生变化 (时间、天气、季节、故事保持不变)
•光线和特效看起来不过度
•它作为一个单一的电影剪辑发挥作用
•超高清晰度。电影质量。
