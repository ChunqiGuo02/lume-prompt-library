# 指示
【[ユーザーが入力]】について、モノクロ＋【1アクセントカラー】のミニマルシアター系縦型ポスターを作成せよ。

# レイアウト要件
・タイトルは画面下部1/4エリアに大きく配置し、和文タイトルを最大サイズ・英文サブタイトルを小さく直下に添える
・上部中央にキャッチコピー（短文1〜2行）を配置する
・左上コーナーに受賞歴・公式認定バッジを縦積みで3〜4点並べる
・右端縦方向に縦書きキャッチコピー（2〜3行）を配置する
・画面中央やや左寄りにメインキャラクターを後ろ姿でシルエット的に立たせる
・背景には幻想的な要素（浮遊する島・巨大生物）をシルエットで表現する
・最下部に制作スタッフクレジット・公開日・配給情報を小さく横並びで配置する
・公開日は右下に大きなフォントで目立たせる

# デザイン要件
・背景はモノクロ（黒〜グレーグラデーション）で統一する
・アクセントカラーは1色のみ（例：鮮やかなコバルトブルーまたはゴールド）
・イラストスタイルはシルエット＋細いラインアートの組み合わせ
・タイトル文字はホワイト＋アクセントカラーで強調する
・全体的にシネマティック・アーティスティックな雰囲気を演出する
・カラフルな背景・過度な装飾・グラデーション多用は避ける
・フォーマット：縦型、アスペクト比 2:3（例：800×1200px）、高解像度

# 指令
为 【[用户输入]】 创建一张极简主义剧院风格的竖版海报，使用单色 + 【一种强调色】。

# 布局要求
- 将标题大字置于底部 1/4 区域，日文标题最大化，英文副标题紧随其下，字号较小。
- 在顶部中心放置一句宣传语（1-2 行）。
- 在左上角垂直堆叠 3-4 个奖项/官方认证徽章。
- 沿右侧边缘放置一句竖排宣传语（2-3 行）。
- 将主角以背影剪影形式置于中心偏左位置。
- 在背景中以剪影形式呈现奇幻元素（浮岛、巨型生物）。
- 在底部以小字水平行排列演职员表、上映日期和发行信息。
- 在右下角用大字体突出显示上映日期。

# 设计要求
- 保持背景为单色（黑色至灰色渐变）。
- 仅使用一种强调色（例如：鲜艳的钴蓝色或金色）。
- 结合剪影和细线条艺术风格。
- 标题文字使用白色 + 强调色进行强调。
- 整体营造电影感和艺术氛围。
- 避免使用彩色背景、过多的装饰或大量的渐变效果。
- 格式：竖版，长宽比 2:3，高分辨率。
