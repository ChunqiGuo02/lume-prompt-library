# 正文 prompt
Turn the main building in the picture into an exploded view, the original composition perspective remains unchanged and the main features of the building are retained. The components of the building are suspended, and the internal structure is clearly visible. Each component presents delicate luster and texture, highlighting the three-dimensional sense and precise details of each component.
