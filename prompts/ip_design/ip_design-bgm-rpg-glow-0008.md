# 正文 prompt
参考画像はリファレンスとしてのみ使用し、スタートフレームには使用しない。バトル風アップテンポBGMあり、効果音あり。開始直後に以下のカットに切り替える。

超高速トランジションで画面切りかえ。高速で空間が崩れ落ちる。

一瞬のホワイトアウトの後、
参考画像の背景の雰囲気を継承したフィールドに切り替わる。

切り替えと同時に、
画面下部にRPG風のコマンドUIが出現する。
そのうちの一つが選択され、キャラクターが動き出す。

UI演出：
・半透明のウィンドウがスライドイン
・ピクセル風またはモダンなゲームUI
・軽く発光する枠（glow）
・カチッとした出現アニメーション

表示内容：
左側：キャラクターの名前、HP、MPゲージ
右側：数種類の参考画像から連想される必殺技(日本語名)
上部：参考画像から連想される敵キャラクター名

参考画像から連想される特殊な巨大な敵キャラクターが右奥に出現し、自キャラは左手前に配置され奥を向いている。

カメラ構図：
・左手前にプレイヤーキャラ
・右奥に敵
・ややローアングルで迫力

演出：
・キャラクターが、その見た目から連想される武器を持ち、戦闘ポーズに入り、敵キャラクターと正対し、軽くステップを踏む。一連の動作は超高速で一瞬で行われる。
・背景は軽く動く（particle, fog, energy）

スタイル：
cinematic, game UI overlay, ultra detailed, high contrast,
fantasy battle scene, dynamic lighting, volumetric light

## 中文译文（源站提供）
参考画像仅作为参照使用，不在起始帧中使用。包含战斗风格的快节奏BGM和音效。在开始后立即切换到以下片段。

以超高速过渡切换画面。空间快速崩塌并坍落。

短暂的白屏之后，
切换到继承参考图背景氛围的场景。

切换时，  
屏幕下方出现 RPG 风格的命令 UI。  
其中一个被选中，角色开始移动。

UI 演出：  
・半透明窗口滑入  
・像素风或现代游戏 UI  
・轻微发光边框（glow）  
・卡嗒声的出现动画

显示内容：  
左侧：角色名称、HP、MP 值条  
右侧：从几张参考图联想到的必杀技（日语名）  
上方：从参考图联想到的敌人角色名  

在右后角出现参考图所联想的特殊巨大敌人，己方角色位于左前方，背对观众。

相机构图：  
・左前方为玩家角色  
・右后方为敌人  
・略低视角呈现冲击力

演出：  
・角色手持与其外貌联想的武器，进入战斗姿势，对准敌人并轻踏步伐。整个动作以超高速在瞬间完成。  
・背景轻微运动（粒子、雾气、能量）

风格：  
电影感、游戏 UI 覆盖、极致细节、高对比度，  
幻想战斗场景，动态照明，体积光。
