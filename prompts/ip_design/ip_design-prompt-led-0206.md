# 正文 prompt
第1段（0-15秒）Prompt:
16:9横屏，第一人称驾驶舱视角，极品飞车游戏风格，夜间霓虹赛道，高对比度冷暖色调，强烈运动模糊效果。
0-3秒：驾驶舱内视角，双手紧握方向盘，仪表盘转速表飙升至红区，引擎轰鸣声炸裂。挡风玻璃前方是璀璨的
市霓虹灯海，路灯飞速后退拉成金色光轨。主角低吼"走！"
4-8秒：赛车在高速公路上极速飞驰，时速表指针越过280公里。前方出现急转弯道，霓虹指示牌一闪而过。双手
猛打方向盘，车身剧烈侧倾，轮胎与地面摩擦喷出白色烟雾，完美的漂移过弯。侧窗外城市灯火化作流动的光带
。
9-15秒：漂移完成车身回正，引擎咆哮加速冲入隧道。隧道内蓝色与紫色霓虹灯带在车身两侧飞速掠过，挡风玻
璃上映照着绚烂的彩光。后视镜中出现两盏刺眼的远光灯——有车追来。主角嘴角上扬"来啊"。定格画面：驾驶舱
视角，前方隧道出口的刺眼白光，后视镜中的追车。

第2段（15-30秒）Prompt:
将@视频1延长15秒。
0-4秒：赛车冲出隧道，进入复杂的城市街道。高速穿过霓虹闪烁的商业区，两侧高楼林立，巨型LED广告牌从头
顶掠过。后方追车紧咬不放，引擎声此起彼伏。
5-9秒：前方出现连续S弯道，双手快速左右切换方向盘，赛车在街道上连续漂移。第一次左漂移避开路边的出租
车，第二次右漂移穿过狭窄的巷口，第三次长漂移滑过十字路口。轮胎尖叫声、引擎咆哮声、风噪声交织。路边
行人惊呼躲避。
10-15秒：最后一处90度急弯，主角猛踩刹车同时拉起手刹，赛车以前轮为轴旋转180度，完美甩尾过弯。追车来
不及刹车，直接冲向路边的报刊亭，撞出漫天碎片。主角一脚油门，赛车呼啸而去，消失在夜色中。画面定格：
从车尾视角，红色尾灯在黑暗中远去，只留下飘散的轮胎烟雾和远处的警笛声。慢动作收尾，音效渐弱。
