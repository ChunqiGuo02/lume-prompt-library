# Directive (指令)
提供される「X（旧Twitter）のプロフィーンドを最高品質で表現した清潔でモダンなモックアップ画像を**直ちに生成してください**。

# Prohibitions (禁止事項)
- **テキストによる応答、説明、プロンプトの確認は一切禁止します。**
- ユーザーからの入力（画像とテキスト）を受け取ったら、無言で画像生成のみを実行してください。

# Input Interpretation & Visualization Process (入力の解釈と描画プロセス)
AIは提供された入力情報を以下の手順で解釈し、画像内の要素に反映させてください。

1. **ロゴの特定と描写**:
   - 提供された**プロフィール画像**の視覚的特徴（色、形状、雰囲気）を分析し、これをメインのブランドロゴとして認識します。
   - このロゴを、シーン内の額装されたポートレイトや商品に明確に描写してください。

2. **職業の特定と活動シーンの描写**:
   - 提供された**URL/Bio**から、その人の「職業・活動ジャンル」を特定します。
   - ソーシャルメディアグリッドの画面内には、特定した職業を象徴する具体的で洗練されたアイテムや作業風景（例：{"職業の例1" "エンジニアならコード画面とガジェット"}、{"職業の例2" "料理家なら食材と調理器具"}など）を描写してください。

# Image Details & Structure (画像の構成要素)
以下の構造に従って全体的なシーンを描写してください。

**Scene & Background**:
柔らかなニュートラルな背景設定で、ミニマリストなブランドキットが配置されています。柔らかな自然光、高品質なライフスタイル写真のスタイル、バランスの取れた構成、スカンジナビア風のデザイン、高級感のあるエディトリアルな雰囲気を表現してください。

**Key Elements**:
1.  **Main Focus (Brand Identity)**:
    - 提供されたプロフィール画像に基づくロゴが、エレガントなタイポグラフィと共に額装されたポートレイトとして明確に展示されています。
2.  **Merchandise**:
    - 同じロゴがプリントされた白いTシャツ、トートバッグ、セラミックマグなどのブランド商品が配置されています。
3.  **Social Media Grid**:
    - タブレットまたはスマートフォン画面上に表示されたグリッドレイアウト。
    - 内容は、前述のプロセスで特定した「職業に関連する活動シーンやアイテムの写真」、舞台裏コンテンツ、引用文、そして暖かいベージュ、クリーム、くすんだテラコッタトーンのソリッドカラーブランドタイルで構成されています。

## 中文译文（源站提供）
# 指令

**立即生成**一张干净、现代、高质量的实体模型图，以展现所提供的 X（前身为 Twitter）个人资料。

# 禁令

- **严禁任何文字回复、解释或提示确认。**
- 收到用户输入（图片和文本）后，只执行图片生成，不发表任何评论。

# 输入解读与可视化过程

AI 应根据以下步骤解读所提供的输入信息，并将其反映在图像元素中。

1.  **标志识别与描绘**：
    -   分析所提供的**个人资料图片**的视觉特征（颜色、形状、氛围），并将其识别为主品牌标志。
    -   在场景中的带框肖像和商品上清晰描绘此标志。

2.  **职业识别与活动场景描绘**：
    -   从所提供的 **URL/简介**中识别人物的“职业/活动类型”。
    -   在社交媒体网格屏幕上，描绘象征所识别职业的特定、精致的物品或工作场景（例如，{"Occupation Example 1" "工程师的代码屏幕和小工具"}，{"Occupation Example 2" "厨师的食材和烹饪用具"}等）。

# 图像细节与结构

根据以下结构描绘整体场景。

**场景与背景**：
柔和、中性的背景设置，配有简约的品牌套件。展现柔和的自然光、高质量的生活方式摄影风格、平衡的构图、斯堪的纳维亚设计和奢华的编辑氛围。

**关键元素**：
1.  **主要焦点（品牌标识）**：
    -   基于所提供的个人资料图片的标志清晰地显示为带有优雅字体的带框肖像。
2.  **商品**：
    -   印有相同标志的品牌商品，如白色 T 恤、手提袋和陶瓷马克杯，整齐排列。
3.  **社交媒体网格**：
    -   显示在平板电脑或智能手机屏幕上的网格布局。
    -   内容包括在先前过程中识别的“与职业相关的活动场景和物品的照片”、幕后内容、语录以及暖米色、奶油色和柔和赤陶色调的纯色品牌瓷砖。
