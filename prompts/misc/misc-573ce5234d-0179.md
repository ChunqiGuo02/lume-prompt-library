# 正文 prompt
生成图片，场景真实，渐变要自然而然的融入式渐变不要分割，【超现实主义时光流转艺术】徽州四季田园诗
32:9超宽全景,展现皖南某徽州古村(宏村风格)的四季生活画卷。
采用村口大樟树下平视角度,可见月沼、祠堂、马头墙、远山。
━━━ 统一元素 ━━━
- 标志性的南湖和画桥贯穿全图
- 同一棵千年香樟树展示完整生命周期
- 祠堂建筑固定,仅内部活动变化
- 同一户三代人的年度生活轨迹
━━━ 冬季:腊月年关 (0-20%)
腊月廿八,村口杀年猪,炊烟袅袅,
马头墙覆薄雪,白墙黑瓦对比强烈,
南湖结薄冰,倒映天空,枯荷残枝,
祠堂准备祭祖,村民贴春联,做冻米糖,
孩子围火炉烤红薯,老人晒太阳唠家常,
水墨画色调,徽墨重彩,传统民俗浓厚,宁静祥和。
━━━ 冬春过渡:立春时节 (20-32%)
- 20%: 除夕夜,祠堂祭祖,灯笼高挂
- 24%: 正月舞龙灯,村民喜庆
- 28%: 雨水节气,冰融水涨,油菜花苞显现
- 32%: 春耕开始,农具准备
━━━ 春季:清明谷雨 (32-45%)
三月油菜花开如金色海洋,映衬白墙黑瓦,
南湖水清,柳绿桃红,画桥上游人如织,
村民采茶制茶,香气飘满村落,
写生的美院学生支起画架,摄影师长枪短炮,
祠堂举办春祭,古礼庄重,
嫩绿金黄色调,诗情画意,春和景明,江南春天典型。
━━━ 春夏过渡:芒种夏至 (45-57%)
- 45%: 油菜收割,金色褪去
- 50%: 插秧季节,水田如镜,倒映天光(中点农忙)
- 54%: 梅雨季节,青石板路湿润,撑油纸伞的身影
- 57%: 荷花初绽,夏意渐浓
━━━ 夏季:小暑大暑 (57-70%)
六月酷暑,南湖荷花盛开,清香远溢,
村民在古树下乘凉,摇蒲扇,孩童湖中戏水,
祠堂举办荷花诗会,文人雅集,
晒秋场晾晒辣椒、笋干,色彩斑斓,
傍晚纳凉,露天放映老电影,
浓绿荷粉色调,生活气息浓郁,夏日慢时光。
━━━ 夏秋过渡:处暑白露 (70-82%)
- 70%: 荷花凋谢,莲蓬采摘
- 75%: 晒秋更盛,屋顶晾晒作物如调色板
- 80%: 香樟树叶渐黄,秋风起
- 82%: 准备中秋,村口搭台唱戏
━━━ 秋季:寒露霜降 (82-100%)
九月晒秋盛景,屋顶晾晒玉米、柿子、辣椒,
如上帝打翻的调色板,航拍必拍,
香樟树金黄,落叶铺满石板路,
南湖秋水澄澈如镜,倒映火红枫叶,
祠堂举办丰收节,村民感恩祖先庇佑,
中秋夜,月沼边赏月,一家团圆,
金黄橙红色调,丰收喜悦,岁月静好,圆满祥和。
━━━ 技术要求 ━━━
- 徽派建筑精准:马头墙、天井、木雕细节考究
- 农事节气对应:每个季节的农活符合真实物候
- 民俗仪式还原:祭祖、舞龙、诗会的场面真实

- 水体镜像强化:南湖倒影是徽州特色,需精细渲染
- 晒秋色彩爆炸:秋季屋顶农作物色彩是视觉高潮
