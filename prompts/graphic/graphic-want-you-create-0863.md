# 正文 prompt
I want you to create a visual story cartoon for this joke
واحد مبوق شاف واحد البنت رقيقة بزاف
سولها: فاش خدامة؟
قالت ليه: عارضة
قال ليها: ف إينا ملعب؟

## 中文译文（源站提供）
我希望你为这个笑话创作一个视觉故事漫画：
一个吸了大麻的家伙看到一个非常苗条的女孩。
他问她：“你做什么工作的？”
她说：“我是模特 (عارضة)。”
他回答说：“哪个体育场的？”（这里是玩文字游戏，因为“模特”在阿拉伯语中也有“体育场主持人”的意思）。
