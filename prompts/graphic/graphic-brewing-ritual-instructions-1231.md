你必须基于该主题，自动构建一个【完整、可信、可被“相信”的视觉体系】。

━━━━━━━━━━━━━━
【核心目标】
━━━━━━━━━━━━━━

生成一张【艺术影盒 + 仪式说明卡】组合的摄影级画面，
看起来像是：
- 被收藏的
- 被研究的
- 被使用过的
- 来自某个完整世界观内部的“实物”

不是装饰画，而是“系统的一部分”。

━━━━━━━━━━━━━━
【自动推导职责（不可省略）】
━━━━━━━━━━━━━━

你必须自行完成以下推导，不向用户提问：

1. 推导该主题所属的【世界类型】
   （例如：童话 / 黑暗童话 / 民俗 / 仪式 / 巫术 / 治愈 / 宗教 / 科幻 / 神秘学 等）

2. 推导对应的：
   - 核心叙事语气（温柔 / 阴森 / 崇高 / 诡秘 / 庄重 / 天真而残酷）
   - 视觉年代感（古老 / 近代 / 不可考）
   - 材质体系（纸张、木头、织物、玻璃、蜡、金属等）

3. 自动生成：
   - 标题名称（符合该世界观语境）
   - 副标题或“信念型文案”
   - 一组“被相信的成分 / 元素 / 道具”
   - 一套说明 / 仪式 / 使用流程文本
   - 所有文字的展示方式（字体气质、位置、是否像咒语/说明/誓言）

- 中央为“主题核心场景”或角色
- 周围陈列：
  - 象征性道具

- 3–5 个步骤或段落：
  - 每一步都有图示 + 文本
  - 文本像是在“指导真实行为”，而不是讲故事
- 带有印章、符号、编号、边角装饰
