# 正文 prompt
Use case: ui-mockup. Create a plausible self-service airport departure kiosk screen photographed at a slight angle in a modern terminal. The screen should show a dense but readable flight-change interface with tabs, seat map preview, boarding pass status, and multilingual alert strip. Exact visible UI text: "Flight Update" "Paris CDG -> Madrid MAD" "Gate B18" "Boarding in 42 min" "English | Francais | Espanol" Design: polished transport UI, high contrast, compact information, realistic reflections on glass. Surroundings: blurred travelers and queue rails in the background. Avoid real airline logos, real airport brands, and watermark.

用例：ui-mockup。创建一个逼真的自助机场登机柜台屏幕截图，该屏幕拍摄于现代航站楼的某个角度。屏幕应显示一个信息丰富但易于阅读的航班变更界面，包括标签页、座位图预览、登机牌状态以及多语言提示信息。屏幕上可见的文字内容如下：“航班更新”“巴黎CDG -> 马德里MAD”“登机口B18”“42分钟后登机”“英语 | 法语 | 西班牙语”。设计要求：界面设计简洁大方、对比度高、信息呈现紧凑，并且玻璃表面应呈现出真实的反射效果。背景中应包含模糊的旅客形象和排队标识。请避免使用真实的航空公司标志、机场品牌标识以及水印。
